üè¶ BKR Credit Check

Step 2: Perform BKR Credit Check

2 Required for all Dutch mortgage applications - checks credit history and negative registrations.

Endpoint: BKR Check

POST /api/dutch-mortgage-qc/bkr-check/{client_id}

Request Body

{ "bsn": "123456789", "consent_given": true }
Important: Client consent is mandatory for BKR checks under Dutch privacy laws (AVG/GDPR).

Response Example

{ "success": true, "bkr_results": { "credit_score": 785, "risk_assessment": "low", "approval_likelihood": "high", "negative_factors": 0, "recommendations": ["Excellent credit profile"] }, "mortgage_impact": { "affects_eligibility": false, "interest_rate_impact": 0.1, "required_actions": [] } }
